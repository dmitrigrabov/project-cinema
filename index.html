    About
    FLIXPLORER

    Flixplorer is a movie search engine app. The purpose of the app is to quickly display the full results of any word search from the Open Movie Database API and allow the user to see more details of any movie shown in the results. It has a responsive layout for mobile, tablet and desktop screen sizes giving an intuative and user-friendly experience.
